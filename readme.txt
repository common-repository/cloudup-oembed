This itsy bitsy plugin adds Cloudup oEmbed support. 

== Description ==

This plugin does what it says on the tin: adds Cloudup oEmbed support. 

== Installation ==

1. Upload the `cloudup-oembed` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Embed happily!
